Lab Programs

This is the repo of all the C++ programs prescribed in BE CSE IV SEM under Bangalore University

For inclusion of "Timer.h" library
1) copy "timer.h" file from (tc -> classlib -> include) and paste it to (tc -> include)
2) and then copy "tclasss.lib" from (tc -> classlib -> lib) and paste it to (tc -> lib)
3) and then (by opening tc) tick mark the container class in (options -> linkers -> libraries)
4) and then (by opening tc) tick mark the default libraries in (option -> linkers -> settings) if it isn't already
